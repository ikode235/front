** 포토샵 하는 법

1. ctrl + n 누르고 
2. width 1920 / height 1000 (높이는 작업하면서 바뀔 수도 있음))-> 둘다 pixels 선택. / 해상도(resolution) 72 선택 / 컬러모드 - RGB /  white
3. view > New Guide Layout (number - 20 / gutter - 20px)
4. f7(레이어) -> 레이어추가 -> view -> Lock guides (이미지 가져올때마다 레이어 추가하기)
5. 세로영역(height) 늘리기 : 
      image > canvas size / relative 체크 / anchor선택 (반대를 체크해야함. 아래를 늘리고싶으면 위 화살표.) / 추가수치 입력 (얼마를 늘릴건지)

- 가이드선 (ctrl + r) (하면 가이드선 나옴 그거 드래그하면 움직이기 가능)
- 크기조정 (ctrl + t) (회전도 가능)
- 선택영역해제 (ctrl + d)
- 상단에  auto-select -> 누르면 알아서 선택됨 (레이어 달라도) / show transform control -> 위에 크기조정하는 툴 알아서 나옴.
- alt + 마우스휠 (확대, 축소), space bar 누른채로 드래그 = 이동
- view > snap 체크 해제? 하면 정확하게 선택? 가능
